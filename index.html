About Draw‑n‑Go

A minimal, focused interface for creating and saving geometric regions. Ideal for geo‑ops, planning tools, or any app that needs quick shape markup on a map.

Templates

Start with ready‑made shapes (square, circle, star, polygon). Adjust corners, radius, and rotation.

Drag & Resize

Accurate handles for moving and scaling, with live readouts for center coordinates and size.

Save Anywhere

Persist regions via REST; export JSON with center (lat,lng) and size. Plug into any backend.

Everything you need to mark a map

  • Snap‑to‑grid and stepwise resizing
  • Undo/redo history
  • Keyboard shortcuts for power users
  • Accessible controls and focus states

Tech Stack

Maps

Supports Google Maps, Mapbox GL JS, or Azure Maps. Swap providers via a small adapter layer.

API

Shape model: { type, center, size, rotation, points[] }. Save as JSON; integrate with any REST backend.

Auth

Add JWT or cookie sessions. Example middleware hooks included in the repo (placeholder).

Performance

Zero‑dependency core UI components; tiny footprint, smooth dragging at 60fps on modest devices.

Integrations

Webhooks on save, optional RT updates (e.g., SignalR / WebSockets) to mirror shapes live to collaborators.

Theming

Customizable tokens and CSS vars for brand colors, spacing, and rounded corners.

How it works

A simple client renders handles around a parametric shape. A controller updates state on drag/resize and emits JSON on save.

1. Select

Choose a template and starting size. Defaults are stored per user.

2. Adjust

Drag the center, scale with corner handles, rotate with the arc handle.

3. Save

Persist or export JSON. Optional webhook notifies downstream systems.
